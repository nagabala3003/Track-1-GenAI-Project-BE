# 🚀 GenAI Workspace API (FastAPI + Gemini + Cloud Run)

## 📌 Overview

This project is a **GenAI-powered backend service** that processes user input using Google Gemini.

It supports multiple AI behaviors such as:

* Summarization
* Question Answering
* Key Points generation

The system uses **prompt engineering + model fallback strategy** to ensure reliable responses.

Deployed on **Google Cloud Run**, it is scalable and serverless.

## ⚙️ Tech Stack

* Backend: FastAPI (Python)
* AI Model: Google Gemini API
* Frontend: React (Vite + TypeScript)
* Server: Uvicorn
* Deployment: Google Cloud Run
* Containerization: Docker

---

## 🧠 Architecture

```
Frontend → FastAPI → Prompt Builder → Gemini API → Response
                        ↓
                 Model Fallback System
```

* Frontend sends user input
* Backend constructs prompt dynamically
* Gemini generates response
* If a model fails → fallback to next model
* Final output returned as JSON

```json
{

  "data": null,
  "error": "Error message details"
}
```

---

## 🤖 Model Selection Strategy

### Default Model

* `gemini-flash-latest`

### Fallback Order

If the primary model fails, the system automatically retries with:

1. `gemini-2.0-flash`
2. `gemini-2.0-flash-lite`
3. `gemini-2.5-flash`

---

## ⚙️ Environment Configuration

```bash
# Required
GEMINI_API_KEY=your_api_key

# Optional (override model priority)
GEMINI_MODEL_CANDIDATES=gemini-flash-latest,gemini-2.0-flash,gemini-2.0-flash-lite,gemini-2.5-flash
```

> If `GEMINI_MODEL_CANDIDATES` is not set, default fallback order is used.
